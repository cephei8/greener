## Supporting Libraries
Libraries to simplify implementing plugins:

- [greener-reporter](https://git.sr.ht/~cephei8/greener-reporter) - native libraries
- [greener-reporter-py](https://git.sr.ht/~cephei8/greener-reporter-py) - Python libraries
- [greener-reporter-js](https://git.sr.ht/~cephei8/greener-reporter-js) - JavaScript libraries
